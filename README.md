# Calculator in C

Simple project made in C that performs multiplication between two numbers entered by the user.

## Features
- Input of two numbers
- Multiplication calculation
- Result display
- Program repetition

## Technologies
- C language

## How to execute
1. Compile the code:
   gcc calculator.c -o calculator

2. Run:
   ./calculator
